# Setup

Setup the npm environment:
```
npm install -i
```

# Run the preview server:
```
hexo server
```

Or with drafts: `hexo server --draft`


# New Blog Post

```bash
# create draft ...
hexo new draft "title"
# edit ...
# convert draft to post ...
hexo publish post "title"
```

# Publish To the Web

`hexo deploy`
